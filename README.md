# Ivanov Basketball Statistics

Ivanov Basketball Statistics is a fan-first basketball analytics experience focused on fast storylines, clean visuals, and mobile-ready insight.

## What It Is

A modern frontend that highlights players, teams, comparisons, and game recaps in a sleek, data-driven layout. The current build uses placeholder data and is ready to be wired to your chosen APIs.

## Roadmap

- Phase 1: Finalize data sources for NBA + EuroLeague.
- Phase 2: Wire player lists, player profiles, team info, standings, and game logs.
- Phase 3: Add interactive charts (trend lines, shot maps, radar comparisons).
- Phase 4: Add alerts, favorites, and shareable stats cards.

## Status

Design and navigation are complete. Data wiring is pending API endpoints.
